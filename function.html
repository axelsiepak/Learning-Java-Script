Function

Defining Functions

In JavaScript, functions allow you to encapsulate code that can be reused. A function is defined using the function keyword followed by a name and parameters:

function greet(name) { return `Hello, ${name}!`; }

Function Expressions

Function expressions involve storing a function in a variable. Unlike function declarations, function expressions are not hoisted.

const greet = function(name) { return `Hello, ${name}!`; };

Arrow Functions

Arrow functions provide a shorter syntax and do not have their own this context, which makes them ideal for callbacks.

const greet = (name) => `Hello, ${name}!`;

Scope and Hoisting

Scope determines the accessibility of variables. JavaScript has both function and block scope. Hoisting refers to the behavior of moving function declarations to the top of their scope.
