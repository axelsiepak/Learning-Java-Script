What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML and XML documents, representing the structure of a web page as a tree of objects that can be manipulated via JavaScript.

Selecting Elements

JavaScript provides methods to select DOM elements, like getElementById and querySelector.

const element = document.getElementById('myElement');

Modifying Elements

Once selected, elements can be modified. This includes changing their content, attributes, and styling.

element.textContent = "New text!";

Event Handling

JavaScript allows you to respond to user interactions by adding event listeners to elements.

element.addEventListener('click', () => alert('Element clicked!'));
